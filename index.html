Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

It is widely used in web development, data analysis, artificial intelligence, and more.

print("Hello, World!")

Key features of Python include:

  • Easy-to-learn syntax
  • Dynamic typing
  • Extensive standard library
Variables and Data Types

Variables are used to store data in Python. You don't need to declare the type of a variable.

x = 10 # Integer y = "Hello" # String

Common data types in Python include:

  • Integers
  • Strings
  • Floats
  • Booleans
Control Flow

Control flow structures allow you to execute code conditionally or repeatedly.

if x > 5: print("x is greater than 5") for i in range(5): print(i)

Common control flow structures include:

  • If statements
  • For loops
  • While loops
Functions

Functions are reusable blocks of code that perform a specific task.

def greet(name): return f"Hello, {name}!"

Functions can take parameters and return values.

Example:

result = greet("Alice") print(result) # Output: Hello, Alice!
Lists and Dictionaries

Lists and dictionaries are two of Python's most commonly used data structures.

my_list = [1, 2, 3] my_dict = {"name": "Alice", "age": 25}

Lists are ordered and mutable, while dictionaries store key-value pairs.

Example:

print(my_list[0]) # Output: 1 print(my_dict["name"]) # Output: Alice
